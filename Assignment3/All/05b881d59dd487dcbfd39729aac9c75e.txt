<section class='article-content'>

<p><a href="http://mashable.com/category/twitter/">Twitter</a>, the <a href="http://www.ebizmba.com/articles/social-networking-websites" target="_blank">second most popular social site in the world</a>, intends to go public on a near-approaching date. That means that if you plan to buy any shares, you'll have a negligible say in the direction the company takes in the future. </p>
<p>We'd like to take this chance to put in our 140 characters. Okay, more like 600 words, in this case.</p>
<p>Though an excellent resource for news, jokes and information, Twitter is imperfect. Below, we've listed eight areas the site could improve upon before and after its IPO.</p>
<div class="see-also"><p>See also: <a href="http://mashable.com/2013/01/14/instagram-changes/" data-crackerjax="#post-slider">11 Things We'd Change About Instagram</a></p></div>
<p>Do you agree with our list? What would you change about Twitter? Make your case in the comments below.</p>
<h2>1. Get rid of conversation lines.</h2>
<p><img src="http://rack.0.mshcdn.com/media/ZgkyMDEzLzA5LzE2Lzc3L2NvbnZlcnNhdGlvLmUwZGJmLmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/83bbf78c/a2f/conversation-lines.jpg" alt="Twitter conversation lines" class="" data-width="1920" data-fragment="m!cb9e" data-image="http://rack.0.mshcdn.com/media/ZgkyMDEzLzA5LzE2Lzc3L2NvbnZlcnNhdGlvLmUwZGJmLmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/83bbf78c/a2f/conversation-lines.jpg" data-micro="1"><small>Image: Screenshot, Twitter</small></p>
<p>Conversation lines, Twitter's late August update, have faced a hugely negative response from users and the press. Twitter likely added the feature as a navigation improvement for newcomers, but it feels like a fundamental shift in the network's purpose. </p>
<blockquote class="twitter-tweet">
<p>Hate, hate, HAAAAAAAATE this new <a href="https://twitter.com/twitter" target="_blank">@Twitter</a> feature that drags old convos back to top of my TL. If I wanted to be on Facebook, I'd be there.</p>
<p>— Clara Jeffery (@ClaraJeffery) <a href="https://twitter.com/ClaraJeffery/statuses/378621189700083712" target="_blank">September 13, 2013</a></p>
</blockquote>
<blockquote class="twitter-tweet">
<p>Congratulations, twitter; the new conversation lines bring old tweets to the top of the timeline, making you about as useful as Facebook.</p>
<p>— netmeg (@netmeg) <a href="https://twitter.com/netmeg/statuses/373173932649500673" target="_blank">August 29, 2013</a></p>
</blockquote>
<p>We'd like to see Twitter put a lid on this experiment and return to the unobtrusive "View Conversation" button.</p>
<h2>2. Scheduled tweets.</h2>
<p>While users can schedule tweets through third party services like <a href="http://mashable.com/category/tweetdeck/">Tweetdeck</a> and <a href="http://mashable.com/category/hootsuite/">Hootsuite</a>, we don't see why Twitter doesn't allow for it on-site. </p>
<h2>3. Show platforms again.</h2>
<p>Twitter no longer displays the platforms from which users posted their tweets. This feature was useful as a platform discovery tool — seeing what tools popular accounts used to manage their social presences. We'd like to see it again.</p>
<h2>4. Access drafts on desktop.</h2>
<blockquote class="twitter-tweet">
<p>hey <a href="https://twitter.com/twitter" target="_blank">@twitter</a> maybe your next update should be so that you can make drafts and quote tweets on the desktop</p>
<p>— please louis ツ (@louisnuggles) <a href="https://twitter.com/louisnuggles/statuses/378121357101264896" target="_blank">September 12, 2013</a></p>
</blockquote>
<p>Drafts: because sometimes the world just isn't ready for your hilarious tweet — yet. The ability to save drafts and access them later is one of the best features of Twitter's mobile app. For some reason, though, drafts are inaccessible on the desktop version. We'd like to see an on-site desktop drafts feature in the future.</p>
<h2>5. Bring back Instagram-friendly Twitter cards.</h2>
<p><img src="http://rack.1.mshcdn.com/media/ZgkyMDEzLzA5LzE2L2NmL3R3aXR0ZXJjYXJkLjNiNzZiLmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/f3a28f38/dd1/twitter-card.jpg" alt="Instagram Twitter card" class="" data-width="1000" data-fragment="m!5d2a" data-image="http://rack.1.mshcdn.com/media/ZgkyMDEzLzA5LzE2L2NmL3R3aXR0ZXJjYXJkLjNiNzZiLmpwZwpwCXRodW1iCTEyMDB4OTYwMD4/f3a28f38/dd1/twitter-card.jpg" data-micro="1"><small>Image: Screenshot, Twitter</small></p>
<p>Remember when <a href="http://mashable.com/category/instagram/">Instagrams</a> automatically displayed when you expanded a tweet? We miss that. </p>
<p>While the cut-off purportedly had nothing to do with <a href="http://mashable.com/category/facebook/">Facebook's</a> purchase of Instagram, Twitter was the only site on which photo displaying was disabled. We'd like the companies to make up, reinstate this feature and beautify our feeds as soon as possible.</p>
<h2>6. Improve location-based searches.</h2>
<p>Twitter's search function has improved exponentially since the company acquired Summize in 2008. However, there is still room for more innovation. One possible path? Allow users to perform better location-based searches. If you're looking for local information, location-based searches would eliminate the need for lengthy, bizarre "event hashtags."</p>
<p><em>Editor's note</em>: While Twitter's advanced search feature allows for a basic location-based searches within 15 miles of a given location, it's not all that easy to use. We'd like to see more search customization, allowing users to narrow or expand the range of their search as they see fit.</p>
<h2>7. Chronology over curation.</h2>
<p>Twitter search results, by default, filter by "top" posts. We're fine with curation as an <em>option</em>, but the huge benefit of Twitter is seeing information chronologically in real time. We want Twitter to make the chronological "all" filter the default choice.</p>
<h2>8. Update "Trends."</h2>
<blockquote class="twitter-tweet">
<p>This is why Trends suck, Twitter hasn't created a sane registration system so all these bots swarm trends with their bullshit</p>
<p>— Sneakernets (@Sneakernets) <a href="https://twitter.com/Sneakernets/statuses/378314599440719872" target="_blank">September 13, 2013</a></p>
</blockquote>
<p>Twitter's "Trends" tool may boast a ton of locations, but it is not at all user-friendly. The only way to view trending topics from different locations is to manually select them individually, each time, from a pop-up screen. We'd like a place for trends in settings, which allows you to select multiple, custom locations at once. That way, we don't  have to continuously switch back and forth.</p>
