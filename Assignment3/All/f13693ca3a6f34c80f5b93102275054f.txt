<h3>Datasets and software used in the papers:
</h3>

<ol>
<li><b>SISAP 2012</b>: Leonid Boytsov, Super-Linear Indices for Approximate Dictionary Searching. SISAP 2012: 162-176
<li><b>JEA ACM 2011</b>: Leonid Boytsov. 2011. Indexing methods for approximate dictionary searching: Comparative analysis. J. Exp. Algorithmics 16, 1, Article 1 (May 2011).
</ol>

You can download the articles <a href="http://boytsov.info/pubs/">on this page.</a>
There are virtually no restrictions on using software and data (see <a href="http://www.apache.org/licenses/LICENSE-2.0">details here</a>) that was designed by me.  I appreciate if you cite my work.
However, the archives contain several third-party packages (which I used for comparison). <b>These packages may be a subject to different licenses and can be guarded by patents</b> (I do believe that agrep and the underlying shift-and algorithm is patented). These packages include at least the following:

<ol>
<li><a href="http://www.dcc.uchile.cl/~gnavarro/">G. Navarro's</a> implementation of the lazy Levenshtein automaton. (The folder NavarroDFA).
<li>NR-grep (developed by G. Navarro).
<li>agrep (developed by Sun Wu and Udi Manber)
<li><a href="http://fastss.csg.uzh.ch/">FastSS</a>
</o>


<h3>Download and build instructions.</h3>

The data and sources used in the JEA ACM 2011 paper are also <a href="http://dl.acm.org/citation.cfm?id=1963191">available here (check the tab "Source Materials").</a>

<ol>
<li>Download and unpack the source file;
<li>Download the datasets to <b>the source file directory</b>;
<li>Check the README file for building/testing instructions.
</ol>
