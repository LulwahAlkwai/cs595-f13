   You can download the articles [1]on this page. There are virtually no
   restrictions on using software and data (see [2]details here) that was
   designed by me. I appreciate if you cite my work. However, the archives
   contain several third-party packages (which I used for comparison).
   These packages may be a subject to different licenses and can be
   guarded by patents (I do believe that agrep and the underlying
   shift-and algorithm is patented). These packages include at least the
   following:
    1. [3]G. Navarro's implementation of the lazy Levenshtein automaton.
       (The folder NavarroDFA).
    2. NR-grep (developed by G. Navarro).
    3. agrep (developed by Sun Wu and Udi Manber)
    4. [4]FastSS

       ACM 2011 paper are also [5]available here (check the tab "Source
       Materials").
         1. Download and unpack the source file;
         2. Download the datasets to the source file directory;
         3. Check the README file for building/testing instructions.
