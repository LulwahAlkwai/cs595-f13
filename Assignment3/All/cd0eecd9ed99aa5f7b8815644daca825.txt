	<h3 class="post-name">IE11 Release Preview for Windows 7: 30% Faster than Other Browsers and Even More Support for Web Standards</h3>
	
    <div class="post-author">
        <span class="avatar"><a href="/3871/ProfileUrlRedirect.ashx"><img src="http://i1.social.s-msft.com/profile/u/avatar.jpg?displayname=ieblog&amp;size=large" alt="" style="border-width:0px;max-height:32px;max-width:32px;" /></a></span>
        
        <span class="profile-usercard-hover" data-profile-userid="3a2c21e8-d203-46d8-9a28-36df8339bc82">
            <span class="user-name"><a class="internal-link view-user-profile" href="/3871/ProfileUrlRedirect.ashx"><span></span>ieblog</a></span>
        
            
        </span>
    </div>
    <div class="post-date">
        <span class="value">
            18 Sep 2013 7:38 AM
        </span>
    </div>
    <div class="post-attributes">
        <div class="attribute-list-header"></div>
        <ul class="attribute-list">
             <li class="attribute-item post-reply-count">
                <span class="attribute-name">Comments</span>
                <span class="attribute-value"><a href="#comments" class="internal-link view-replies"><span></span>89</a></span>
             </li>
        </ul>
        <div class="attribute-list-footer"></div>
    </div>
    <div class="post-content user-defined-markup"><p>Internet Explorer 11 continues to provide the best Web experience that is fast, fluid and perfect for touch, across the full range of Windows devices and screen sizes. The IE11 Release Preview for Windows 7 is <a href="http://go.microsoft.com/fwlink/?LinkID=316880">available for download</a> today and includes performance improvements making it 30% faster than other browsers, updates to support the latest Web standards, and developer tools improvements in response to your feedback on the IE11 <a href="http://blogs.msdn.com/b/ie/archive/2013/07/25/ie11-developer-preview-for-windows-7-enabling-next-generation-sites-and-apps-faster.aspx">Developer Preview</a> for Windows 7.</p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><iframe src="http://www.youtube.com/embed/S97t6uEY8BI" frameborder="0" width="640" height="480"></iframe></p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><em>This video shows some of the performance and developer tools enhancements in IE11 Release Preview for Windows 7</em></p>
<p>With IE11 on Windows 7, customers receive a fast, secure browser that is compatible with existing Web sites and delivers exciting new interactive Web experiences. For developers, IE11 supports the latest Web standards and technologies, making it easier to build sites that work across devices. The new F12 tools in IE11 enable developers to build high performance Web experiences, faster and more efficiently.</p>
<p><strong>A Faster Web Experience</strong></p>
<p>Performance matters to everyone who browses the Web. The IE11 Release Preview for Windows 7 includes additional performance tuning so that real-world sites download and display fast and are highly responsive and interactive.</p>
<p>You can experience IE11&rsquo;s leading performance first hand with demos such as <a href="http://ie.microsoft.com/testdrive/Performance/Levitation/">Levitation</a>, <a href="http://ie.microsoft.com/testdrive/Performance/LawnMark/">Lawnmark</a>, and <a href="http://ie.microsoft.com/testdrive/Performance/LiteBrite/">Lite-Brite</a> on the <a href="http://www.ietestdrive.com/">IE Test Drive site</a>, showing real world Web site patterns for graphically rich, interactive, and 3D experiences. In honor of tomorrow's <a href="http://en.wikipedia.org/wiki/International_Talk_Like_a_Pirate_Day">International Talk Like a Pirate Day</a>, our newest test drive demo, <a href="http://ie.microsoft.com/testdrive/performance/pirates/">PirateMarrrk</a>, tests your browser&rsquo;s HTML5, JavaScript, and Scalable Vector Graphics performance.</p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><img style="max-width: 95%; border: 1px solid #999;" src="http://blogs.msdn.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-38-71-metablogapi/6787.irpfwftobaemsfws_2D00_image1_5F00_760x427.png" alt="The new PirateMarrrk test drive demo tests your browser&rsquo;s HTML5, JavaScript, and Scalable Vector Graphics performance." /></p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><em>The new </em><a href="http://ie.microsoft.com/testdrive/performance/pirates/"><em>PirateMarrrk</em></a><em> test drive demo tests your browser&rsquo;s HTML5, JavaScript, and Scalable Vector Graphics performance.</em></p>
<p>Internet Explorer 11 advances the performance leadership of our JavaScript engine, Chakra, while ensuring compatibility, interoperability, and security. As we improve performance for real-world sites, IE11&rsquo;s JavaScript performance on benchmarks like WebKit SunSpider continues to lead compared with latest version of other browsers. On Windows 7, IE11 Release Preview is 9% faster than IE10, which is a 5% improvement from the IE11 Developer Preview, and over 30% faster than the nearest competitive browser.</p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><img style="max-width: 95%; border: 1px solid #999;" src="http://blogs.msdn.com/cfs-filesystemfile.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-38-71-images/4380.IE11RPWin7SunSpider.png" alt="IE11 extends its leadership in Javascript performance, so real-world sites are faster" /></p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><em>IE11 extends its leadership in </em><a href="http://ie.microsoft.com/testdrive/benchmarks/SunSpider/Default.html"><em>Javascript performance</em></a><em>, so real-world sites are faster</em></p>
<p><strong>Even More Web Standards and Increased Compatibility with Existing Web Sites</strong></p>
<p>IE11 Release Preview for Windows 7 includes updates to reflect the latest emerging Web standards. For example, now that the <a href="http://www.w3.org/TR/pointerevents/">Pointer Events</a> specification is a Candidate Recommendation at the W3C, IE11 supports an un-prefixed version of the emerging standard. With Pointer Events support across the full range of Windows devices (and soon to other browsers), Web sites can easily build experiences that work equally well with mouse, keyboard, pen, and touch.</p>
<p>IE11 Release Preview for Windows 7 adds new user controls for the <a href="http://blogs.msdn.com/b/ie/archive/2013/04/25/enabling-professional-quality-online-video-new-specifications-for-interoperable-captioning.aspx">Standard Delivery Profile for Closed Captioning</a>. With the new control, accessible from the Internet Options menu, users can customize how captions appear in the browser, even overriding the default styling provided by the video source. This customization further advances IE11 as the best browser for <a href="http://blogs.msdn.com/b/ie/archive/2013/09/05/online-professional-quality-video-premium-media-experiences-without-plug-ins-in-internet-explorer-11.aspx">professional-quality online video without plugins</a>.</p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><img style="max-width: 95%; border: 1px solid #999;" src="http://blogs.msdn.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-38-71-metablogapi/4380.irpfwftobaemsfws_2D00_image3.png" alt="With IE11, users can customize the appearance of closed captions, enabling an improved online video experience" /></p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><em>With IE11, users can customize the appearance of closed captions, enabling an improved online video experience</em></p>
<p>The IE11 Release Preview for Windows 7 also supports the latest <a href="http://www.w3.org/2011/tracking-protection/drafts/tracking-dnt.html">Tracking Preference Expression (DNT)</a> draft in the W3C. After installation, the IE11 Release Preview prompts the user set a default tracking preference that Internet Explorer will send to Web sites. On upgrade, IE11 Release Preview automatically preserves the user&rsquo;s previous tracking preferences. .</p>
<p><strong>Faster, More Efficient Developer Tools Productivity, Based on Your Feedback</strong></p>
<p>IE11 includes a completely re-designed and enhanced suite of in-browser F12 developer tools. These tools help Web developers diagnose and optimize their apps quickly and efficiently. Having fast and reliable Web apps is more critical than ever.</p>
<p style="margin-bottom: 0;">The IE11 Release Preview for Windows 7 includes several tools enhancements to increase developer productivity and in response to the overwhelming enthusiasm and feedback we have received from developers. Here are just a few of the many improvements:</p>
<ul style="margin-top: 0;">
<li><strong>Console: </strong>IE11 makes it easier to debug WebGL content by mapping WebGL errors to the corresponding JavaScript location. Developer productivity is improved because primitives have been added to Intellisense. With IE11, developers can debug multi-threaded Web sites with full console access while targeting Web workers.</li>
<li><strong>Debugger: </strong>An improved file picker allows faster access to files in a complex Web project.</li>
<li><strong>DOM Explorer: </strong>With IE11, developers can tune Web sites more quickly by editing CSS property values and seeing those edits in the Live DOM without having to press the enter key</li>
<li><strong>Memory: </strong>To make it easier to narrow down the source of memory leaks, the F12 tool now identifies what operations were performed on allocated data between memory snapshots.</li>
</ul>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><img style="max-width: 95%; border: 1px solid #999;" src="http://blogs.msdn.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-38-71-metablogapi/7433.irpfwftobaemsfws_2D00_image4_5F00_760x404.png" alt="The IE11 Memory tool identifies which memory was allocated, de-allocated, or modified between snapshots, supporting faster memory tuning and diagnosis" /></p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><em>The IE11 Memory tool identifies which memory was allocated, de-allocated, or modified between snapshots, supporting faster memory tuning and diagnosis</em></p>
<ul>
<li><strong>Networking: </strong>A new &ldquo;always refresh from server&rdquo; option creates reproduceable performance results while developers rapidly iterate to tune their site performance. The tool also includes new Search Next and Search Previous functions for analyzing complex network traces.</li>
</ul>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><img style="max-width: 95%; border: 1px solid #999;" src="http://blogs.msdn.com/cfs-file.ashx/__key/communityserver-blogs-components-weblogfiles/00-00-00-38-71-metablogapi/4278.irpfwftobaemsfws_2D00_image5_5F00_760x191.png" alt="With IE11, the &ldquo;always refresh from server&rdquo; option supports iterative development by producing reproducible network performance results each time you reload the page" /></p>
<p style="text-align: center; font-size: 8.25pt; font-style: italic;"><em>With IE11, the &ldquo;always refresh from server&rdquo; option supports iterative development by producing reproducible network performance results each time you reload the page</em></p>
<p><strong>A Better Web Today, and Ahead</strong></p>
<p>IE11 provides the best Web experience across the full range of Windows devices and screen sizes with mouse, keyboard, and especially touch. Try it out for yourself with <a href="http://blogs.msdn.com/b/stevengu/archive/2013/09/09/download-windows-8-1-rtm-visual-studio-2013-rc-and-windows-server-2012-r2-rtm-today.aspx">Windows 8.1 RTM</a> from MSDN, and if you are a Windows 7 user, <a href="http://go.microsoft.com/fwlink/?LinkID=316880">download the IE11 Release Preview</a> today. Virtual Machine images of IE11 Release Preview for Windows 7 will also be available on <a href="http://modern.ie">modern.ie</a> later this week.</p>
<p>Now is the time for developers to confirm that your Web sites are ready for the hundreds of millions of Windows devices and customers moving to IE11 and for business customers to verify your readiness to deploy IE11. You can find a full list of new functionality available to developers in the IE11 Developer Guide <a href="http://msdn.microsoft.com/en-us/library/ie/bg182636(v=vs.85).aspx">here</a>.</p>
<p>We look forward to your feedback. Please share your suggestions on <a href="http://connect.microsoft.com/ie">Connect</a>.</p>
<p>Sandeep Singhal and Rob Mauceri<br />Group Program Managers, Internet Explorer</p><div style="clear:both;"></div></div>
