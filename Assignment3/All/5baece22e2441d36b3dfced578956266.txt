											<h2 class="entry-title"><a href="http://gilkalai.wordpress.com/2013/07/06/some-old-and-new-problems-in-combinatorics-and-geometry/" rel="bookmark">Some old and new problems in combinatorics and&nbsp;geometry</a></h2>					
					<div class="entry-meta">
													<span class="meta-prep meta-prep-author">Posted on</span> <a href="http://gilkalai.wordpress.com/2013/07/06/some-old-and-new-problems-in-combinatorics-and-geometry/" title="9:00 am" rel="bookmark"><span class="entry-date">July 6, 2013</span></a>							<span class="by-author"><span class="sep">by</span> <span class="author vcard"><a class="url fn n" href="http://gilkalai.wordpress.com/author/gilkalai/" title="View all posts by Gil Kalai" rel="author">Gil Kalai</a></span> </span>											</div><!-- .entry-meta -->

					<div class="entry-content">
						<h2><a href="http://gilkalai.files.wordpress.com/2013/07/paul99.jpg"><img class="alignnone size-full wp-image-10529" alt="Paul99" src="http://gilkalai.files.wordpress.com/2013/07/paul99.jpg?w=640"   /></a></h2>
<p><strong><span style="color:#ff0000;">Paul Erdős in Jerusalem, <del>1933</del>  1993</span></strong></p>
<p>I just came back from a great <a href="http://www.renyi.hu/conferences/erdos100/">Erdős Centennial conference</a> in wonderful Budapest. I gave a lecture on old and new problems (mainly) in combinatorics and geometry (<a href="http://gilkalai.files.wordpress.com/2013/07/erdos100-20.pdf">here are the slides</a>), where I presented twenty problems, here they are:</p>
<h2>Around Borsuk&#8217;s Problem</h2>
<p>Let <img src='http://s0.wp.com/latex.php?latex=f%28d%29&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='f(d)' title='f(d)' class='latex' /> be the smallest integer so that every set of diameter <span style="line-height:1.5;">one in <img src='http://s0.wp.com/latex.php?latex=R%5Ed&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='R^d' title='R^d' class='latex' /> can be covered by <img src='http://s0.wp.com/latex.php?latex=f%28d%29&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='f(d)' title='f(d)' class='latex' /> sets of smaller diameter. Borsuk conjectured that <img src='http://s0.wp.com/latex.php?latex=f%28d%29+%5Cle+d%2B1&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='f(d) &#92;le d+1' title='f(d) &#92;le d+1' class='latex' />.</span></p>
<p>It is known (Kahn and Kalai, 1993) that : <img src='http://s0.wp.com/latex.php?latex=f%28d%29+%5Cge+1.2%5E%7B%5Csqrt+d%7D&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='f(d) &#92;ge 1.2^{&#92;sqrt d}' title='f(d) &#92;ge 1.2^{&#92;sqrt d}' class='latex' />, <span style="line-height:1.5;">and also that (Schramm, 1989) <img src='http://s0.wp.com/latex.php?latex=f%28d%29+%5Cle+%28%5Csqrt%7B3%2F2%7D%2Bo%281%29%29%5Ed&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='f(d) &#92;le (&#92;sqrt{3/2}+o(1))^d' title='f(d) &#92;le (&#92;sqrt{3/2}+o(1))^d' class='latex' />.</span></p>
<p><span style="color:#800080;"><strong>Problem 1:</strong></span> Is <em>f(d)</em> exponential in <em>d</em>?</p>
<p><span style="color:#800080;"><strong>Problem 2:</strong></span> What is the smallest dimension for which Borsuk&#8217;s conjecture is false?</p>
<h2>Volume of sets of constant width in high dimensions</h2>
<p><span style="color:#800080;"><strong>Problem 3:</strong></span> Let us denote the volume of the n-ball of radius 1/2 by <img src='http://s0.wp.com/latex.php?latex=V_n&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='V_n' title='V_n' class='latex' />.</p>
<p><strong>Question</strong> (Oded Schramm): Is there some <img src='http://s0.wp.com/latex.php?latex=%5Cepsilon+%3E0&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='&#92;epsilon &gt;0' title='&#92;epsilon &gt;0' class='latex' /> so that for every <img src='http://s0.wp.com/latex.php?latex=n%3E1&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='n&gt;1' title='n&gt;1' class='latex' /> there exist a set <img src='http://s0.wp.com/latex.php?latex=K_n&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='K_n' title='K_n' class='latex' /> of constant width 1 in dimension n whose volume satisfies <img src='http://s0.wp.com/latex.php?latex=VOL%28K_n%29+%5Cle+%281-%5Cepsilon%29%5En+V_n&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='VOL(K_n) &#92;le (1-&#92;epsilon)^n V_n' title='VOL(K_n) &#92;le (1-&#92;epsilon)^n V_n' class='latex' />.</p>
<h2>Around Tverberg&#8217;s theorem</h2>
<p>Tverberg&#8217;s Theorem states the following: Let <img src='http://s0.wp.com/latex.php?latex=x_1%2Cx_2%2C%5Cdots%2C+x_m&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='x_1,x_2,&#92;dots, x_m' title='x_1,x_2,&#92;dots, x_m' class='latex' /> be points in <span style="line-height:1.5;"><img src='http://s0.wp.com/latex.php?latex=R%5Ed&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='R^d' title='R^d' class='latex' /> with <img src='http://s0.wp.com/latex.php?latex=m+%5Cge+%28r-1%29%28d%2B1%29%2B1&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='m &#92;ge (r-1)(d+1)+1' title='m &#92;ge (r-1)(d+1)+1' class='latex' />. </span><span style="line-height:1.5;">Then there is a partition <img src='http://s0.wp.com/latex.php?latex=S_1%2CS_2%2C%5Cdots%2C+S_r&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='S_1,S_2,&#92;dots, S_r' title='S_1,S_2,&#92;dots, S_r' class='latex' /> of <img src='http://s0.wp.com/latex.php?latex=%5C%7B1%2C2%2C%5Cdots%2Cm%5C%7D&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='&#92;{1,2,&#92;dots,m&#92;}' title='&#92;{1,2,&#92;dots,m&#92;}' class='latex' /> such that  </span><span style="line-height:1.5;"><img src='http://s0.wp.com/latex.php?latex=%5Ccap+_%7Bj%3D1%7D%5Erconv+%28x_i%3A+i+%5Cin+S_j%29+%5Cne+%5Cemptyset&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='&#92;cap _{j=1}^rconv (x_i: i &#92;in S_j) &#92;ne &#92;emptyset' title='&#92;cap _{j=1}^rconv (x_i: i &#92;in S_j) &#92;ne &#92;emptyset' class='latex' />.</span></p>

<h2>F ≤ 4E</h2>
<p><span style="color:#800080;"><strong>Problem 9:</strong> </span>Let <em>K</em> be a two-dimensional simplicial complex and suppose that <em>K</em> can be embedded in <img src='http://s0.wp.com/latex.php?latex=R%5E4&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='R^4' title='R^4' class='latex' />. Denote by <em>E</em> the number of edges of <em>K</em> and by <em>F</em> the number of 2-faces of K.</p>
<p><strong>Conjecture: </strong><em><strong>F <strong>≤</strong> 4E</strong></em></p>
<p>A weaker version which is also widely open and very interesting is: For some absolute constant <em>C</em>, <strong><em>F <strong>≤</strong> C E.</em></strong></p>

<h2>Expectation thresholds and thresholds</h2>
<p><span style="color:#800080;"><strong>Problem 16:</strong></span> Consider a random graph <em>G</em> in <strong>G(n,p)</strong> and the graph property: <em>G</em> contains a copy of a specific graph <em>H</em>. <h2>Conditions for colorability</h2>
<p><strong><span style="color:#800080;">Problem 19:</span> </strong>A conjecture by Roy Meshulam and me:</p>
<p>There is a constant <em>C</em> such that every graph <em>G</em><br />
with no induced cycles of order divisible by 3 is colorable by <em>C</em> colors.</p>
<p><span style="color:#800080;"><strong>Problem 20:</strong></span></p>
<p>Another conjecture by Roy Meshulam and me: For every <em>b&gt;0</em> there<br />
is a constant <em>C=C(b)</em> with the following property:</p>
<p>Let <em>G</em> be a graph such that for all its induced subgraphs <em>H</em></p>
<blockquote><p><em>The number of independent sets of odd size minus the number of independent sets of even size is between </em>-b<em>  <span style="line-height:1.5;">and </span></em><span style="line-height:1.5;">b</span><em><span style="line-height:1.5;">.</span></em></p></blockquote>
<p>Then <em>G</em> is colorable by <em>C(b)</em> colors.</p>
<h2>Remarks:</h2>
<p>The title of the lecture is borrowed from several papers and talks by Erdős.<span id="more-10519"></span> (And I tried to imitate a little his lectures.)  The presentation give some more details on the twenty problems, and here are a few further comments. I devoted a few posts to questions around Borsuk&#8217;s problem (<a href="http://gilkalai.wordpress.com/2011/06/30/around-borsuks-conjecture-1-some-problems/">I</a>, <a href="http://gilkalai.wordpress.com/2011/06/15/the-combinatorics-of-cocycles-and-borsuks-problem/">II</a>, <a href="http://gilkalai.wordpress.com/2012/07/23/a-weak-form-of-borsuk-conjecture/">III</a>), and an <a href="http://mathoverflow.net/questions/29000/volumes-of-sets-of-constant-width-in-high-dimensions">MO question</a> for problem 3 on sets of constant width in high dimensions. I also <a href="http://gilkalai.wordpress.com/2008/12/23/seven-problems-around-tverbergs-theorem/">devoted a post</a> to questions around Tverberg&#8217;s theorem where problems 4 and 5 are mentioned. I asked problems 5 and 6 in the early 70&#8242;s. Problem 7 is motivated by a very special case of problem 5. Problem 8 is recent and was <a href="http://mathoverflow.net/questions/114646/intersecting-family-of-triangulations">asked also on MO</a>. Gjergji Zaimi (private communication) proposed a far-reaching extension:</p>
<p><strong style="line-height:1.5;">Question:</strong><span style="line-height:1.5;"> Let <img src='http://s0.wp.com/latex.php?latex=P&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='P' title='P' class='latex' /> be a convex polytope without triangular two dimsneional faces. Is it true that a collection of vertices so that every two of them belongs to a facet, has at most as many vertices as the largest facet?</span></p>
<p>Problem 9 which represents one of my current main research effort was asked by Sarkaria, me, and perhaps others in the late 80s. (See also <a href="http://gilkalai.wordpress.com/2013/02/01/f-4e/">this post</a>.) Problem 10 was the subject of polymath 3 that started <a href="http://gilkalai.wordpress.com/2010/09/29/polymath-3-polynomial-hirsch-conjecture/">here</a>. Problem 11 intrigued me since my undergraduate years, <a href="http://gilkalai.wordpress.com/2008/08/20/two-very-early-problems-a-simple-solution-and-a-new-problem/">see this post</a>, and I recalled it again after hearing a lecture by Haim Brezis. (See <a href="http://www.math.rutgers.edu/~brezis/publications.html">Haim&#8217;s publication list</a> for some relevant papers with Bourgain,  Mironescu, Nguyen, and others.) Problems 12 and 13 are motivated by recent progress on Möbius randomness and on finding primes in various sets of integers, and also by <a href="http://mathoverflow.net/questions/79113/why-so-difficult-to-prove-infinitely-many-restricted-primes">this MO problem</a>. Problem 14 was motivated by a lecture of Bollobas and by my general interest in fault-tolerance see this <a href="http://cstheory.stackexchange.com/questions/17914/does-a-noisy-version-of-conways-game-of-life-support-universal-computation">TCSoverfow problem</a> and this <a href="http://mathoverflow.net/questions/132402/conways-game-of-life-for-random-initial-position">MO problem</a>. For open problems on polytopes including Problems 10, and 15 see <a href="http://gilkalai.wordpress.com/2008/05/07/five-open-problems-regarding-convex-polytopes/">this post</a>.  Problems 16, 17 also represent one of my main current research efforts, see <a href="http://front.math.ucdavis.edu/0603.5218">this paper with Jeff Kahn</a> for problem 16 and <a href="http://gilkalai.files.wordpress.com/2013/07/chor8-4.pdf">this paper with Kahn</a> on problem 17. (The best known very recent upper bound by Bourgain fot problem 17 is not yet published.)  Variants of problem 18 were raised by several other people (Körner, Katona, Gowers). Problems 19 and 20 are motivated by Helly-type questions and in particular by the fractional Helly property.</p>
<p>More comments: (The post was launched a day too early before it was completed, so let me add a few more comments.)</p>
<p>The solution of Borsuk&#8217;s problem was a very nice success to Erdős&#8217; mathematics. It relies on the interplay between combinatorial and geometric structures. In fact, Erdős himself, Rogers, Danzer, and perhaps others suggested that combinatorial configurations will lead to counterexamples. Larman made a crucial step in translating the question from geometry to combinatorics. Larman also asked about 2-distance sets and this was crucial to the recent counterexample in dimension 65. (See <a href="http://gilkalai.wordpress.com/2013/05/27/andriy-bondarenko-showed-that-borsuks-conjecture-is-false-for-dimensions-greater-than-65/">this post</a>.) Problem 7 is related to a special case of Conjecture 5. It is based on replacing a directed edge from vertex<em> i</em> to vertex <em>j</em> by the point <img src='http://s0.wp.com/latex.php?latex=e_i-e_j&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='e_i-e_j' title='e_i-e_j' class='latex' />. Problem 15 is known only for <img src='http://s0.wp.com/latex.php?latex=k%3D2&amp;bg=ffffff&amp;fg=333333&amp;s=0' alt='k=2' title='k=2' class='latex' />. It is true for zonotopes (where a simplex face is not an option) with <em>d=2k</em>.  This case is a high-dimensional analog of Gallai-Sylvester theorem. It is quite interesting for simple polytopes.</p>

			function JetpackLikesWidgetQueueHandler() {
				var wrapperID;
				if ( ! jetpackLikesMasterReady ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				if ( jetpackLikesWidgetQueue.length > 0 ) {
					// We may have a widget that needs creating now
					var found = false;
					while( jetpackLikesWidgetQueue.length > 0 ) {
						// Grab the first member of the queue that isn't already loading.
						wrapperID = jetpackLikesWidgetQueue.splice( 0, 1 )[0];
						if ( jQuery( '#' + wrapperID ).hasClass( 'jetpack-likes-widget-unloaded' ) ) {
							found = true;
							break;
						}
					}
					if ( ! found ) {
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				} else if ( jQuery( 'div.jetpack-likes-widget-unloaded' ).length > 0 ) {
					// Get the next unloaded widget
					wrapperID = jQuery( 'div.jetpack-likes-widget-unloaded' ).first()[0].id;
					if ( ! wrapperID ) {
						// Everything is currently loaded
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				}

				var $wrapper = jQuery( '#' + wrapperID );
				$wrapper.find( 'iframe' ).remove();

				if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "<iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='22px' width='68px' frameBorder='0' scrolling='no' src='" + $wrapper.data( 'src' ) + "'></iframe>" );
				} else {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "<iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='55px' width='100%' frameBorder='0' src='" + $wrapper.data( 'src' ) + "'></iframe>" );
				}

				$wrapper.removeClass( 'jetpack-likes-widget-unloaded' ).addClass( 'jetpack-likes-widget-loading' );

				$wrapper.find( 'iframe' ).load( function( e ) {
					var $iframe = jQuery( e.target );
					$wrapper.removeClass( 'jetpack-likes-widget-loading' ).addClass( 'jetpack-likes-widget-loaded' );

					JetpackLikespostMessage( { event: 'loadLikeWidget', name: $iframe.attr( 'name' ), width: $iframe.width() }, window.frames[ 'likes-master' ] );

					if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
						$wrapper.find( 'iframe' ).Jetpack( 'resizeable' );
					}
				});
				setTimeout( JetpackLikesWidgetQueueHandler, 250 );
